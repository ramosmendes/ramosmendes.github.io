Workspring - Sistema de Gerenciamento de Workshops

Java Spring Boot JPA PostgreSQL RESTful API

Este projeto visa otimizar o gerenciamento e a inscrição em workshops, que antes podiam ser manuais ou fragmentados. Desenvolvi um **sistema completo** focado na eficiência e escalabilidade.

  • API RESTful robusta com endpoints **CRUD** para criação, visualização, atualização e exclusão de workshops e participantes.
  • Utilização de **Spring Boot 3** para um desenvolvimento ágil e integração com **JPA** para persistência de dados em um banco de dados **PostgreSQL**.
  • Implementação de validações de dados e tratamento de exceções para garantir a **integridade** e a **segurança** das informações.
  • **Resultado:** Proporciona uma plataforma centralizada e de fácil uso, melhorando a organização e acessibilidade dos workshops.

LogSpring - Sistema de Autenticação e Autorização

Java Spring Security JWT Autenticação Segura

Criei este sistema como uma solução de autenticação e autorização reutilizável para aplicações Java, abordando a necessidade de **segurança robusta** e **controle de acesso granular** em ambientes de desenvolvimento.

  • Desenvolvimento de um sistema de autenticação seguro utilizando **Spring Security** para proteção de recursos.
  • Implementação de **JSON Web Tokens (JWT)** para a geração e validação de tokens de acesso, garantindo a comunicação segura.
  • Configuração personalizada de regras de segurança e controle de acesso baseado em **roles de usuário**.
  • **Resultado:** Garante um ambiente seguro para o acesso de usuários, facilitando a integração com outras aplicações e padronizando o processo de autenticação.
